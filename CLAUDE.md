# Project Rules

## `manuscripts/` folder — read-only

The `manuscripts/` folder contains Dave's actual prose drafts (e.g. `manuscripts/book 1 - what wasn't and what was/`). This is distinct from `docs/`, which is worldbuilding/canon documentation and is fair game for direct edits.

- **Read** files in `manuscripts/` freely.
- **Never** use Write, Edit, or any delete operation inside `manuscripts/` — not even if asked to apply a fix directly.
- If asked for feedback or improvements, describe them as suggestions in chat. Dave applies any changes himself.
